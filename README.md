# My Website
### 主要使用： HTML / CSS / javascript / jQuery
### 其他輔助工具：PageSpeed Insights/ lozad / TinyPNG / Webfont
### 功能說明：
* 使用jQuery plugin製作捲動動畫。
* 使用CSS和SVG製作動畫。
* 使用jQuery製作簡易的雙語切換。
* 部署在Netlify上。
* 與Netlify form聯動，當使用者透過表單聯絡時，Netlify會自動寄信通知。
* 使用 Google檢測工具檢測網站速度。
* 使用lozad懶載入工具，降低FCP。
* 使用tinyPNG壓縮圖檔。
